# Hand-gesture-presentation-system
A real-time Hand Gesture Presentation System that allows users to control presentation slides using hand gestures with Computer Vision and Python. Built using OpenCV and MediaPipe for touchless and interactive presentations.

# Features
- Slide navigation using gestures
- Real-time hand tracking
- Touchless presentation control
- Interactive user experience

# Technologies Used
- Python
- OpenCV
- MediaPipe
- CVZone

# How to Run
1. Install required libraries:
pip install opencv-python mediapipe cvzone

2. Run the project:
python main.py
